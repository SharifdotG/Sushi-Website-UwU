# Sushi Website UwU

Responsive sushi-themed landing page built with HTML, modular CSS, Vite, and AOS.

## Live Demo

[sushi-website-uwu-dotg.vercel.app](https://sushi-website-uwu-dotg.vercel.app)

## Overview

This project recreates and adapts the sushi website from the JavaScript Mastery tutorial, with a Vite-based setup that is ready for local development and Vercel deployment.

Tutorial source:

- Video: [Creating a Sushi-Themed Website: HTML & CSS Guide for Beginners](https://www.youtube.com/watch?v=QRrPE9aj3wI)
- Original repository: [adrianhajdin/project_html_css_website](https://github.com/adrianhajdin/project_html_css_website)

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vite
- AOS

## Features

- Responsive landing page layout with dedicated sections for hero, about, popular foods, trending items, subscription, and footer
- Modular stylesheet structure under `css/sections`
- Scroll-triggered animations powered by AOS for below-the-fold sections
- Dedicated load-time intro animations for the header and hero content
- Static-site friendly setup for Vercel deployment

## Getting Started

### Prerequisites

- Node.js
- npm

### Install

```bash
npm install
```

## Install AOS

```bash
npm install aos
```

### Run Locally

```bash
npm run dev
```
